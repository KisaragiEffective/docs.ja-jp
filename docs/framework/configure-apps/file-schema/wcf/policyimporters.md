# \<policyImporters>

この構成要素は、バインディングに関するカスタム ポリシー アサーションのインポートを制御するすべてのポリシー インポーターを指定します。 各子要素は、 `policyImporter` バインディング機能に関するカスタムポリシーアサーションの検索、およびアサーションに必要な機能を実装するカスタムバインド要素のアタッチに使用される <> です。
