# <a name="querying-a-conceptual-model"></a>概念モデルに対するクエリ

ADO.NET Entity Framework を使用すると概念モデルに対してクエリを実行できます。 Entity Framework の最新版を使用して概念モデルに対してクエリを実行するには、「[データのクエリ](/ef/ef6/querying/index)」を参照してください。
