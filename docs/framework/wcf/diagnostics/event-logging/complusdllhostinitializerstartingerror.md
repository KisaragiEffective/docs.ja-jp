# <a name="complusdllhostinitializerstartingerror"></a>ComPlusDllHostInitializerStartingError

Id: 131  
  
 重大度 : エラー  
  
 カテゴリ : ServiceModel  
  
## <a name="description"></a>説明  

 このイベントは、DllHost 初期化子の開始時にエラーが発生したことを示します。 イベントには、AppID、例外、プロセス名、およびプロセス ID が表示されます。
