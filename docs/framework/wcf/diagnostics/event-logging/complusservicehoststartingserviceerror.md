# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError

ID : 130  
  
 重大度 : エラー  
  
 カテゴリ : ServiceModel  
  
## <a name="description"></a>説明  

 このイベントは、COM+ サービスの開始時にエラーが発生したことを示します。 イベントには、AppID、CLSID、サロゲート、例外、プロセス名、およびプロセス ID が表示されます。
