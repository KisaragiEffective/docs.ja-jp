# <a name="sslnoaccessibleprivatekey"></a>SslNoAccessiblePrivateKey

Id: 155  
  
 重大度 : エラー  
  
 カテゴリ : TransactionBridge  
  
## <a name="description"></a>説明  

 このイベントは、特定のサブジェクト名と拇印を持つ ID 証明書にアクセス可能な秘密キーがないことを示します。 イベントには、プロセス名とプロセス ID が表示されます。
