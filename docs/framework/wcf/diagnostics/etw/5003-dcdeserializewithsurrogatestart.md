|||  
|-|-|  
|id|5003|  
|Keywords|シリアル化|  
|Level|"詳細"|

## <a name="description"></a>説明  

 このイベントは、DataContract のサロゲートによる逆シリアル化を開始すると生成されます。  
  
## <a name="message"></a>Message  

 DataContract のサロゲートによる %1 のシリアル化解除を開始します。
