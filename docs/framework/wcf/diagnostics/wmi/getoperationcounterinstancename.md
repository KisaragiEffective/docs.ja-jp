# <a name="getoperationcounterinstancename"></a>GetOperationCounterInstanceName

操作パフォーマンス カウンターのインスタンスの名前を取得します。  
  
## <a name="syntax"></a>構文  
  
```csharp
string GetOperationCounterInstanceName(  
  string Operation  
);  
```  
  
## <a name="parameters"></a>パラメーター  
  
### <a name="operation"></a>Operation  

 [in] string
