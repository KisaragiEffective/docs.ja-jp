# <a name="imetadataemitsetmoduleprops-method"></a>IMetaDataEmit::SetModuleProps メソッド

[IMetaDataEmit::D efinemoduleref](imetadataemit-definemoduleref-method.md)の前の呼び出しで定義されているモジュールへの参照を更新します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT SetModuleProps (
    [in]  LPCWSTR     szName  
);  
```  
  
## <a name="parameters"></a>パラメーター  

 `szName`  
 からUnicode のモジュール名。 これはファイル名のみで、完全なパス名ではありません。
