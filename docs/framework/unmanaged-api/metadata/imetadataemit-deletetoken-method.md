# <a name="imetadataemitdeletetoken-method"></a>IMetaDataEmit::DeleteToken メソッド

現在のメタデータスコープから指定されたトークンを削除します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT DeleteToken (
    [in]  mdToken     tkObj
);  
```  
  
## <a name="parameters"></a>パラメーター  

 `tkObj`  
 から削除するトークン。
