# <a name="iceegengetstring-method"></a>ICeeGen::GetString メソッド

指定した相対仮想アドレスに格納されている文字列を取得します。  
  
 このメソッドは互換性のために残されています。使用しないでください。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetString (  
    [in]  ULONG      RVA,
    [out] LPWSTR     *lpString  
);  
```  
  
## <a name="parameters"></a>パラメーター  

 `RVA`  
 から返される文字列の相対仮想アドレス。  
  
 `lpString`  
 入出力返された文字列。
