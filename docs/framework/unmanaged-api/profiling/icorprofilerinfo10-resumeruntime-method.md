# <a name="icorprofilerinfo10resumeruntime-method"></a>ICorProfilerInfo10:: ResumeRuntime メソッド

GC を実行せずにランタイムを再開します。

## <a name="syntax"></a>構文

```cpp
HRESULT ResumeRuntime();
```
