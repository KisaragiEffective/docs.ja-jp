# <a name="icordebugevalisactive-method"></a>ICorDebugEval::IsActive メソッド
この、このオブジェクトが現在実行されているかどうかを示す値を取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT IsActive (  
    [out] BOOL              *pbActive  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `pbActive`  
 入出力この評価がアクティブかどうかを示す値へのポインター。
