# <a name="icordebugframegetchain-method"></a>ICorDebugFrame::GetChain メソッド
このフレームが含まれているチェーンへのポインターを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetChain (  
    [out] ICorDebugChain     **ppChain  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `ppChain`  
 入出力このフレームを含むチェーンを表す、のオブジェクトのアドレスへのポインター。
