# <a name="icordebugarrayvaluegetcount-method"></a>ICorDebugArrayValue::GetCount メソッド
配列内の要素の合計数を取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT GetCount (  
    [out] ULONG32 *pnCount  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `pnCount`  
 入出力配列内の要素の合計数へのポインター。
