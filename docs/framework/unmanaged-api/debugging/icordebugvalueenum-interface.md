# <a name="icordebugvalueenum-interface"></a>ICorDebugValueEnum インターフェイス

"ICorDebugEnum" メソッドを実装し、"ICorDebugValue" 配列を列挙します。  
  
## <a name="methods"></a>メソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[次のメソッド](icordebugvalueenum-next-method.md)|現在の位置から開始して、指定した数の `ICorDebugValue` インスタンスを列挙から取得します。|  
  
## <a name="remarks"></a>解説  
  
> [!NOTE]
> このインターフェイスは、コンピューター間またはプロセス間でのリモート呼び出しをサポートしていません。
