# <a name="itypenamebuilderaddassemblyspec-method"></a>ITypeNameBuilder::AddAssemblySpec メソッド

## <a name="syntax"></a>構文  
  
```cpp  
HRESULT AddAssemblySpec (  
    [in] LPCWSTR szAssemblySpec  
);  
```
