# <a name="itypenamebuilderopengenericargument-method"></a>ITypeNameBuilder::OpenGenericArgument メソッド

このメソッドは、.NET Framework インフラストラクチャをサポートします。独自に作成したコードから直接使用するためのものではありません。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT OpenGenericArgument ( );  
```
