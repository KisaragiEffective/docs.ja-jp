# <a name="iobjecthandle-interface"></a>IObjectHandle インターフェイス

値渡しのマーシャリングオブジェクトを間接参照からラップ解除するためのメソッドを提供します。  
  
## <a name="methods"></a>メソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[Unwrap メソッド](iobjecthandle-unwrap-method.md)|間接参照から値渡しのマーシャリングオブジェクトをラップ解除します。|
