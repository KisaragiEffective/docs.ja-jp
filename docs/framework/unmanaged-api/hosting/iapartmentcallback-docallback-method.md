# <a name="iapartmentcallbackdocallback-method"></a>IApartmentCallback::DoCallback メソッド
アパートメント内で指定された関数を実行します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT _stdcall DoCallback(  
    [in] SIZE_T pFunc,  
    [in] SIZE_T pData  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `pFunc`  
 からアパートメント内で実行される関数へのポインター。  
  
 `pData`  
 から関数の引数へのポインター。
