# <a name="iassemblynamefinalize-method"></a>IAssemblyName::Finalize メソッド

この [IAssemblyName](iassemblyname-interface.md) オブジェクトが、デストラクターが呼び出される前にリソースを解放し、その他のクリーンアップ操作を実行できるようにします。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT Finalize ();  
```
