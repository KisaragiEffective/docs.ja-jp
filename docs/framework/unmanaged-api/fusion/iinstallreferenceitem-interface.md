# <a name="iinstallreferenceitem-interface"></a>IInstallReferenceItem インターフェイス

|メソッド|説明|  
|------------|-----------------|  
|[GetReference メソッド](iinstallreferenceitem-getreference-method.md)|この `IInstallReferenceItem`によって表される `FUSION_INSTALL_REFERENCE` 構造体へのポインターを取得します。|
