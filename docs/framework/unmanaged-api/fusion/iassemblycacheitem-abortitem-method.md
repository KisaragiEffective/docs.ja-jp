# <a name="iassemblycacheitemabortitem-method"></a>IAssemblyCacheItem::AbortItem メソッド
アセンブリが解放される前に、グローバルアセンブリキャッシュ内のアセンブリでクリーンアップ操作を実行できるようにします。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT AbortItem ();  
```
