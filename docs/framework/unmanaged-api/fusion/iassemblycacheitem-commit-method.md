# <a name="iassemblycacheitemcommit-method"></a>IAssemblyCacheItem::Commit メソッド
キャッシュされたアセンブリ参照をメモリにコミットします。  
  
## <a name="syntax"></a>構文  
  
```cpp  
HRESULT Commit (  
    [in] DWORD dwFlags,   
    [out, optional] ULONG *pulDisposition  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `dwFlags`  
 からFusion に定義されているフラグ。  
  
 `pulDisposition`  
 [out、省略可能]操作の結果を示す値です。
