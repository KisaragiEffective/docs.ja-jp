# <a name="search-text-not-found"></a>検索文字列が見つかりませんでした

指定したテキストが見つかりませんでした。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 指定したテキストのスペルが正しいことを確認してください。
