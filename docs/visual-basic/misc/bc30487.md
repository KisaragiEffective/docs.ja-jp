# <a name="operator-operatorname-is-not-defined-for-type-typename"></a>演算子 '\<operatorname>' は、型 '\<typename>' に対して定義されていません

`+`などの型に対して単項演算子 ( `-`、 `Not`、または `String`) を使用しようとしましたが、これは定義されていません。  
  
 **エラー ID:** BC30487  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 単項演算子を式の適切な演算子に置き換えます。
