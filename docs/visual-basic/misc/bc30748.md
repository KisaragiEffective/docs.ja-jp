# <a name="cannot-convert-to-type"></a>' ' に変換できません。 \<type>

無効な変換を行おうとしました。  
  
 **エラー ID:** BC30748  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 互換性のある変換の種類を選択します。
