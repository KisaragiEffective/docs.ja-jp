# <a name="argument-argumentname-is-not-valid-for-the-array"></a>引数 '\<argumentname>' は配列には無効です。

配列には、無効な引数が含まれています。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 式の引数のスペルを確認します。 変数名のスペルが間違っていると、0 に初期化される数値型の変数が暗黙的に生成されることがあります。
