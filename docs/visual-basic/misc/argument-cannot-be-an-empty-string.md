# <a name="argument-cannot-be-an-empty-string"></a>引数を空の文字列にすることはできません

引数が、空の文字列 ("") として定義されています。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 引数に対して有効な値を指定します。
