# <a name="types-declared-private-must-be-inside-another-type"></a>'Private' と宣言された型は、別の型の内部になければなりません

`Private` 修飾子が別の型の内部にない型で使用されました。  
  
 **エラー ID:** BC31089  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. `Friend`などの制限の緩いアクセス修飾子を使用します。  
  
2. 別の型の内部で型を宣言します。
