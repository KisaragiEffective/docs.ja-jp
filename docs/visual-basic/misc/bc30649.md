# <a name="typename-is-an-unsupported-type"></a>'\<typename>' はサポートされていない型です

Visual Studio では、指定された型を表すことができません。  
  
 **エラー ID:** BC30649  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- サポートされる型を使用します。  
  
- 新しい型を開発します。
