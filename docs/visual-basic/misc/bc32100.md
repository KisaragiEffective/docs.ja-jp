# <a name="as-comma-or--expected"></a>'As'、コンマ、または ')' が必要です。

指定されている型パラメーターに `As`、コンマ (`,`)、または終わりかっこ ( `)` ) がありません。  
  
 **エラー ID:** BC32100  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 不足している `As`、コンマ、または終わりかっこを指定します。
