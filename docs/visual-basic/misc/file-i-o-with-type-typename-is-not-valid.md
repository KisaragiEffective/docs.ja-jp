# <a name="file-io-with-type-typename-is-not-valid"></a>型 '\<typename>' のファイル I/O が有効ではありません

サポートされていない型をファイルに書き込もうとしています。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 構造体を確認して、サポートされていない型が検出された場所を判断し、有効な型に変更します。
