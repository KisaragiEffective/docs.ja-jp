# <a name="inappropriate-use-of-keyword-keyword-in-a-module"></a>不適切な使用の\<キーワード > モジュール内のキーワード
モジュールはインスタンスを持たず、継承をサポートしたり、インターフェイスを実装したりしません。 そのため、次のキーワードはモジュールの宣言に適用されません。  
  
- [MustInherit](../../visual-basic/language-reference/modifiers/mustinherit.md)  
  
- [NotInheritable](../../visual-basic/language-reference/modifiers/notinheritable.md)  
  
- [Overloads](../../visual-basic/language-reference/modifiers/overloads.md)  
  
- [Private](../../visual-basic/language-reference/modifiers/private.md)  
  
- [Protected](../../visual-basic/language-reference/modifiers/protected.md)  
  
- [Shadows](../../visual-basic/language-reference/modifiers/shadows.md)  
  
- [Shared](../../visual-basic/language-reference/modifiers/shared.md)  
  
- [Static](../../visual-basic/language-reference/modifiers/static.md)  
  
 サポートされる唯一のキーワード、[Module ステートメント](../../visual-basic/language-reference/statements/module-statement.md) は [Public](../../visual-basic/language-reference/modifiers/public.md) と [Friend](../../visual-basic/language-reference/modifiers/friend.md) します。  
  
 さらに、使用することはできません、[Implements 句](../../visual-basic/language-reference/statements/implements-clause.md)ステートメントまたは[Inherits ステートメント](../../visual-basic/language-reference/statements/inherits-statement.md)モジュールのステートメント ブロックにします。  
  
 既定では、このメッセージは警告です。 警告を表示しない方法や、警告をエラーとして扱う方法の詳細については、「[Visual Basic での警告の構成](/visualstudio/ide/configuring-warnings-in-visual-basic)」を参照してください。  
  
 **エラー ID:** BC42028  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- このプログラミング要素をモジュールにする場合は、その宣言内で `Public` キーワードまたは `Friend` キーワードのみを使用します。 モジュールのアクセス レベルを指定しない場合、既定では `Friend` が使用されます。  
  
- このプログラミング要素のインスタンスを作成する場合は、クラスとして宣言します。 そうすると、クラス宣言に適用されるキーワードを使用できます。
