# <a name="argument-value-pathname-contains-characters-that-are-not-valid-in-a-path-name"></a>引数値 '\<pathname>' は、パス名に有効でない文字を含んでいます。

パス名に無効な文字が 1 つ以上含まれています。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- パス名から無効な文字を削除します。
