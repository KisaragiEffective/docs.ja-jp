# <a name="operator-operator-must-have-one-parameter"></a>演算子 '\<operator>' には、1 つのパラメーターが必要です
単項演算子がパラメーターなしで定義されているか、2 つ以上のパラメーターを使用して定義されています。  
  
 単項演算子には、必ず 1 つのパラメーターを指定する必要があります。  
  
 **エラー ID:** BC33014  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 1 つのパラメーターを指定するように、定義を調整します。  
  
- 2 つのパラメーターを指定する必要がある場合は、二項演算子を定義する必要があります。  
  
- パラメーターが不要または 2 つ以上必要な場合は、、[Function ステートメント](../../visual-basic/language-reference/statements/function-statement.md)を使用して、演算子の代わりに `Function` プロシージャを定義する必要があります。
