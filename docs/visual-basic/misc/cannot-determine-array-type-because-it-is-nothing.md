# <a name="cannot-determine-array-type-because-it-is-nothing"></a>配列が Nothing であるため、配列型を指定できません

`Nothing`の値に設定されているため、配列の型を判断できません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 配列に `Nothing` 以外の値を指定します。
