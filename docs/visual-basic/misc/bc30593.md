# <a name="variables-in-modules-cannot-be-declared-specifier"></a>モジュール内の変数を ' ' として宣言することはできません。 \<specifier>

`MustInherit`ステートメント内の変数に `Module` などの指定子を使用しました。 モジュールは、インスタンス化することができず、継承をサポートしません。また、インターフェイスを実装することもできません。  
  
 **エラー ID:** BC30593  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 指定子を削除します。
