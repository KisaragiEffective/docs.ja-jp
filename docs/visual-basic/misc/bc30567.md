# <a name="array-initializer-is-missing-number-elements"></a>配列初期化子には \<number> 個の要素が足りません。

配列の初期化子で指定した要素が少なすぎます。  
  
 **エラー ID:** BC30567  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 不足している引数を指定します。  
  
- `ReDim` を使って配列のサイズを変更します。
