# <a name="bad-guid-format"></a>不適切な GUID 形式です

GUID が、正しくない形式で指定されています。  
  
 **エラー ID:** BC42035  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- GUID の形式を確認します。
