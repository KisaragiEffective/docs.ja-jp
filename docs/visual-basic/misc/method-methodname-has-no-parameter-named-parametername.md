# <a name="method-methodname-has-no-parameter-named-parametername"></a>メソッド '\<methodname>' に '\<parametername>' という名前のパラメーターがありません

指定された名前付き引数が対象メソッドの引数のいずれにも一致しません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 名前付き引数が正しく指定されていることを確認します。
