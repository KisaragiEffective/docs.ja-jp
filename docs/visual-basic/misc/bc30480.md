# <a name="shared-sub-new-cannot-be-declared-specifier"></a>共有 'Sub New' を '\<specifier>' として宣言することはできません

`Shared Sub New` 宣言で有効ではない指定子を使用しようとしました。  
  
 **エラー ID:** BC30480  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 指定子を削除します。
