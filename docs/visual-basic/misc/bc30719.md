# <a name="case-statements-are-not-valid-in-the-immediate-window"></a>'Case' ステートメントは、イミディエイト ウィンドウでは有効ではありません

`Case` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30719  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- `Case` イミディエイト **ウィンドウから** ステートメントを削除します。
