# <a name="static-local-variable-variablename-is-already-declared"></a>スタティック ローカル変数 '\<variablename>' は、既に宣言されています。

同じ名前の別の静的ローカル変数が既に宣言されています。  
  
 **エラー ID:** BC31401  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 冗長な静的ローカル宣言を削除します。  
  
2. 各静的ローカル変数に一意の名前を付けます。
