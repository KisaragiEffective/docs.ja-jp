# <a name="unable-to-parse-xml-error"></a>データを解析できません: \<error>

使用中の XML ファイルにエラーがあります。  
  
 **エラー ID:** BC31023  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- XML が整形式であること、有効であること、ファイルが破損していないことを確認します。
