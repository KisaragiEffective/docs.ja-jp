# <a name="target-folder-is-a-file"></a>ターゲット フォルダーがファイルです

フォルダー関連のアクションを実行しようとしているときにファイルを指定しています。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 指定しているパスを確認して、ファイルではなくフォルダーを指定するようにします。
