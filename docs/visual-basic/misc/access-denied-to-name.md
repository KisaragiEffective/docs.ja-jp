# <a name="access-denied-to-name"></a>'\<name>' へのアクセスが拒否されました

コンピューターにおけるセキュリティ アクセス許可が不十分であるため、デバイス、ファイル、ディスクにアクセスできません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 必要なアクセス許可があることを確認します。
