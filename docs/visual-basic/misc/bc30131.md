# <a name="security-attribute-attributename-cannot-be-applied-to-a-module"></a>セキュリティ属性 '\<attributename>' をモジュールに適用することはできません。

共通言語ランタイムでは、モジュールのセキュリティ属性は受け入れられません。

**エラー ID:** BC30131

## <a name="to-correct-this-error"></a>このエラーを解決するには

- 宣言から属性を削除します。
