# <a name="design-guidelines-for-exceptions"></a>例外のデザインのガイドライン

例外処理には、戻り値に基づくエラー報告よりも多くの利点があります。 優れたフレームワーク設計により、アプリケーション開発者は例外の利点を実感できます。 このセクションでは、例外の利点について説明し、それらを効果的に使用するためのガイドラインを示します。
