# <a name="compiler-error-cs1035"></a>コンパイラ エラー CS1035

ファイルの終わりが見つかりました。'*/' が必要です。  
  
 開始コメント区切り記号と終了区切り記号が一致しませんでした。  
  
 次の例では CS1035 が生成されます。  
  
```csharp  
// CS1035.cs  
public class a  
{  
   public static void Main()  
   {  
   }  
}  
/*   // CS1035, needs closing comment  
```
