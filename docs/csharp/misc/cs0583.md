# <a name="compiler-error-cs0583"></a>コンパイラ エラー CS0583

内部コンパイラ エラー コンパイラで内部エラーが発生しました。 この問題を回避するには、以下にリストされている場所付近のプログラムを単純化するか変更してください。 一覧の上部の場所は、内部エラーが発生したポイントに近い場所です。

> [!NOTE]
> このコンパイラ エラーは、Roslyn では使用されなくなりました。
