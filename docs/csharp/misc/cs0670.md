# <a name="compiler-error-cs0670"></a>コンパイラ エラー CS0670

フィールドは void 型を持てません。  
  
 フィールドが [void](../language-reference/builtin-types/void.md)型になるように宣言されました。  
  
 次の例では CS0670 が生成されます。  
  
```csharp  
// CS0670.cs  
class C  
{  
   void f;   // CS0670  
   // try the following line instead  
   // public int f;  
  
   public static void Main()  
   {  
      C myc = new C();  
      myc.f = 0;  
   }  
}  
```
