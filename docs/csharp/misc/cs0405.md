# <a name="compiler-error-cs0405"></a>コンパイラ エラー CS0405

型パラメーター 'type parameter' に対する制約 'constraint' が重複しています。  
  
 ジェネリック宣言に対する 2 つの制約が同じです。 エラーを解消するには、重複を削除します。  
  
 次の例では CS0405 が生成されます。  
  
```csharp  
// CS0405.cs  
interface I  
{  
}  
  
class C<T> where T : I, I  // CS0405.cs  
{  
}  
```
