# <a name="compiler-error-cs1629"></a>コンパイラ エラー CS1629

アンセーフ コードは反復子には記述できません。  
  
 C# 言語仕様は、反復子に安全でないコードを許可していません。  
  
 次の例では CS1629 が生成されます。  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe
using System.Collections.Generic;  
class C
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
