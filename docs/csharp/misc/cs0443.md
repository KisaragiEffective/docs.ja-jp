# <a name="compiler-error-cs0443"></a>コンパイラ エラー CS0443

構文エラーです。値が必要です  
  
 このエラーは、配列インデックスの値を指定せずに配列を参照するときに発生します。  
  
## <a name="example"></a>例  

 次のコードでは CS0443 が生成されます。  
  
```csharp  
// CS0443.cs
using System;
class MyClass
{  
    public static void Main()
    {  
        int[,] x = new int[1,5];  
        if (x[] == 5) {} // CS0443  
        // if (x[0, 0] == 5) {}
    }  
}  
```
