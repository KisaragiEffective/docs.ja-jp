# <a name="compiler-error-cs0574"></a>コンパイラ エラー CS0574

デストラクターの名前をクラスの名前と同じにしてください。  
  
 デストラクターの名前は、クラス名の前にチルダ (~) を付けたものである必要があります。  
  
 次の例では CS0574 が生成されます。  
  
```csharp  
// CS0574.cs  
namespace x  
{  
   public class iii  
   {  
      ~iiii()   // CS0574  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
