# <a name="compiler-error-cs0723"></a>コンパイラ エラー CS0723

スタティック型 'type' の変数を宣言することはできません  
  
 スタティック型のインスタンスは作成できません。  
  
 次の例では CS0723 が生成されます。  
  
```csharp  
// CS0723.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC sc = null;  // CS0723  
   }  
}  
```
