# <a name="compiler-error-cs0568"></a>コンパイラ エラー CS0568

構造体に明示的なパラメーターのないコンストラクターを含めることはできません。  
  
 各[構造体](../language-reference/builtin-types/struct.md)には、既にオブジェクトをゼロに初期化するパラメーターなしコンストラクターがあります。 そのため、構造体に対して作成するコンストラクターは、1 つ以上のパラメーターを受け取る必要があります。  
  
 次の例では CS0568 が生成されます。  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
